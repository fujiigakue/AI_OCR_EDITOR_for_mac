# AI_OCR_EDITOR_for_mac
令和大蔵経プロジェクトエディターです。
ブラウザ上での編集作業を代替します。
WindowsPCをお使いの方はこちらからダウンロードしてください。

---

## 📦 インストール方法

### 前提条件
以下の環境が必要です。
* OS: Windows 10/11 (64bit 推奨)
* ブラウザエンジン: Microsoft Edge WebView2 Runtime がインストールされていること（※最新のWindowsでは標準搭載されています）

### 手順
1. **「Releases」 > 「v1.0.〇」(最新のリリース) > 「AI_OCR_EDITOR.zip」 を選択**
2. **ダウンロードしたZipファイルを右クリック > 「すべて展開」　を選択**

※以下のコマンドでもダウンロードできます。
   ```bash
   git clone https://github.com/fujiigakue/AI_OCR_EDITOR_for_Windows.git
   ```

## 💻 使用方法

### 表示方法
1. **AI_OCR_EDITOR.exeをクリック**
2. **ブラウザでGoogleログインを求められるため、ログイン**
3. **編集を行う経典ページに遷移し、そのページのURLをコピーする。**
4. **エディター画面上部のURL入力欄に編集したいページのURLを貼り付けする。**
5. **「比較開始」ボタンを押下する。**
6. **上段に大正蔵、下段にAI-OCRで読み取られた文字列、左部にIIIFの画像が表示される。**<br>
*※二回目以降の利用では、最近修正したページのURLが自動入力されます。*<br>
*※もし以前作業したページをもう一度作業したい場合は、「履歴」から選択することが可能です。*<br>

### 編集方法
1. **AI-OCRの各行下部の編集アイコンをクリックする。（Ctrl + Space）**
2. **編集したい行が青枠で囲われたら、編集を行う。**
3. **編集が完了したら、「修正を送信」ボタンを押下する。（Ctrl + s)**
4. **削除したい場合は、青枠で囲われた後「重複のため削除」ボタンを押下する。(Ctrl + d)**
5. **ノートへ移動にしたい場合は、青枠で囲まれた後「ノートへ移動」ボタンを押下する。（Ctrl + n)**
6. **編集が完了したら少しの時間緑枠で表示されます。**<br>
*※もし編集中に想定外のスクロールが生じた場合、「赤枠の位置まで移動」（Shift）を押下すると赤枠の位置まで戻れます。*<br>

### 編集に便利な機能
* 大正蔵とAI-OCRで異なる漢字が使われている箇所は黄色で網掛けされます。クリックすると縦の列がそろいます。
* 大正蔵の網掛け部分の漢字をクリックするだけで、クリップボードにその漢字がコピーされます。
* 頻出漢字登録機能(漢字を選択して右クリック→「頻出漢字登録をおこなう」、「頻出漢字リスト表示」→該当漢字押下)
* SAT2018検索機能（漢字を選択て右クリック→「SAT2018で検索する」）
* CHISE IDS 異字検索機能（漢字を選択して右クリック→「CHISE IDSで異字検索する」）
* しおり機能（AI-OCR表示欄で該当する行を選択して右クリック→「しおりをはさむ」、経典読み込み完了時にダイアログが表示され遷移可能）
* 履歴機能（「履歴」→該当するURLをクリック→「比較開始」押下）
* 文字拡大縮小機能（Ctrl + マウスホイール）

### ショートカット一覧
* Ctrl + Space：赤枠の行を編集モードに変更する。
* Ctrl + s：「修正を送信する」ボタンを押下する。
* Ctrl + d：「重複のため削除」ボタンを押下する。
* Ctrl + n：「ノートへ移動」ボタンを押下する。
* Ctrl + r：「閲覧版再生成」ボタンを押下する。
* Shift：「赤枠の位置まで移動」ボタンを押下する。
